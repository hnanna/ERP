# ERP Implementation – Production Module (Cocoon)

## Mô tả
Dự án tập trung vào triển khai phân hệ Sản xuất (Production Module) trong hệ thống ERP cho doanh nghiệp mỹ phẩm Cocoon.  
Nội dung bao gồm phân tích quy trình sản xuất, xây dựng tài liệu nghiệp vụ và thiết kế hệ thống nhằm hỗ trợ quản lý sản xuất hiệu quả.

## Nội dung chính
- Phân tích quy trình sản xuất
- Xây dựng BOM (Bill of Materials)
- Thiết kế quy trình sản xuất trên ERP
- Tài liệu hóa nghiệp vụ và sơ đồ hệ thống
